#Мини-калькулятор на Express.js
Проект представляет из себя веб-приложение калькулятора на Express.js. В приложении используются принципы работы с Express.js, сервер, маршруты, middleware и обработка запросов.

#Функционал
* Базовые операции калькулятора: Сложение, вычитание, умножение, деление
* Два типа запросов: GET с query-параметрами и POST с JSON телом
* Проверка входных данных и обработка возможных ошибок
* middleware для отслеживания запросов
* HTML страница для тестирования API
* frontend часть через Express

#Запуск и работа с проектом

1.Запуск - node src/app.js

2.Перейдите по адресу: http://localhost:3000

3.Запрос GET

a - первое число
b - второе число
op - операция (add, sub, mul, div)
Пример запроса - GET /api/calc?a=10&b=5&op=add

Пример тела запроса JSON
{
  "a": 10,
  "b": 5,
  "op": "add"
}
